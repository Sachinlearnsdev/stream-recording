Every time you click Import Bundle, your current OBS state gets copied to %APPDATA%\obs-studio\_clip-prep-backup-<timestamp>\ before the new bundle is unpacked. If the import breaks something, click Restore on a backup below to roll back to exactly how OBS was before that import.

▶ Danger zone
▶
Normal uninstall — keeps your files
Removes auto-start (Win login), Start Menu + Desktop shortcuts, and the clip-prep:// URL protocol. Your install dir, OBS scenes, recordings, and keys all stay on disk. Re-run bootstrap to wire it back up.
🚨 Full reset — deletes EVERYTHING
Removes the install dir, OBS Studio app + config (scenes/profiles/plugins), all registry entries, all shortcuts. Keeps your local repo + recordings folder + Node/git/ffmpeg installs. Use this before a clean re-iex when something's stuck.
⚠ Export your OBS bundle first if you want your scenes back after reinstall — Dashboard → Export OBS Bundle, save the zip OUTSIDE %LOCALAPPDATA%.
Why no button? The watcher process lives inside the install dir — it can't delete itself reliably. Paste the command below into a fresh PowerShell window. It asks for explicit NUKE confirmation, kills the watcher + OBS, deletes everything, and prints the iex one-liner for reinstall.
System
Keys & Channels
All keys stored locally in sasi-secrets.js at your install dir, gitignored. Plain text — only your Windows user can read it. The Go Live tab's Channel switcher uses what you configure here.
Shared · StreamElements JWTs
One SE account usually covers all your channels, so these JWTs are shared. Powers alerts (super chats, members, gifts, redemptions).
Channels
Each channel has its own brand (logo / name / tagline), YouTube channel + API keys, Twitch username, and preferred theme. YT API keys are per-channel — the system needs the right key for the right channel ID to poll chat. You can re-use the same API key string across channels if you only have one Google Cloud key.

0 keys · ~0h coverage

Twitch

Client ID + Secret are reserved for future Twitch-API features (follower events, channel points). Register an app at dev.twitch.tv/console if you want to use them.
