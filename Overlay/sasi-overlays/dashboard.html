▶ OBS Bundle (export · import · backups)
▶
A bundle is a portable folder with all your OBS state — scenes, profiles, plugin settings, plugin DLLs. Export to backup, Import to restore on a new PC.
Close OBS before either operation — it locks browser-source state.
Already done during install. Run again if you create a new scene collection.
Auto-backups (rollback safety)
Every time you click Import Bundle, your current OBS state gets copied to %APPDATA%\obs-studio\_clip-prep-backup-<timestamp>\ before the new bundle is unpacked. If the import breaks something, click Restore on a backup below to roll back to exactly how OBS was before that import.
